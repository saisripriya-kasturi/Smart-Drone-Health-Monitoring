# Day 5 – Data Acquisition System with ESP32

## Objectives

Developing a data acquisition system based on ESP32 and sending data about drone’s health parameters to the cloud dashboard.

## Drone Health Parameters

* Battery level: 95%
* Temperature: 35°C
* Altitude: 120m

## Tasks Accomplished

* Created a new thingSpeak channel
* Configured various fields for drone’s health parameters
* Connected ESP32 to Wi-Fi network in Wokwi
* Sent battery, temperature and altitude parameters data to ThingSpeak
* Represented the data graphically using ThingSpeak graphs

## Learning Outcomes

* Learnt what data acquisition system is
* Learnt how to send several parameters’ data through ESP32
* Experienced with ThingSpeak cloud system
* Learnt how to visualize sensor data using dashboards and graphs

## Conclusion

Successfully created a simulated data acquisition system based on ESP32 and sent the data about drone health parameters to ThingSpeak cloud platform.
