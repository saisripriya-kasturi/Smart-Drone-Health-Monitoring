# Day 13 - Telemetry Data Export Implementation

## Objective
To implement data export functionality for the Smart Drone Health Monitoring System and prepare telemetry data for analysis and documentation.

## Work Done

- Exported drone telemetry data from ThingSpeak cloud platform.
- Generated CSV file containing recorded drone parameters.
- Verified exported data values with the monitoring dashboard.
- Organized telemetry data for further analysis and final project documentation.
- Prepared data records including battery level, temperature, altitude, motor status, GPS location, and flight duration.

## Data Export

The exported CSV file contains:

- Battery Voltage
- Temperature
- Altitude
- Motor Status
- Latitude
- Longitude
- Flight Duration
