# Day 9 - Flight Duration Monitoring and Alert System


## Objective
To implement flight duration tracking and integrate time-based alert functionality into the ESP32-based drone monitoring system.


## Work Done
- Implemented flight duration monitoring using time-based programming.
- Added a timer mechanism to calculate the total operating time of the drone.
- Added flight duration as an additional monitoring parameter.
- Integrated flight duration data into the ThingSpeak cloud dashboard.
- Implemented a flight time limit alert to improve drone safety monitoring.
- Tested and verified the continuous update of flight duration values.


## Outcome
- Successfully added automatic flight duration tracking to the drone monitoring system.
- The system can monitor operating time and generate alerts when the defined limit is exceeded.
